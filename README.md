# EEC 289Q Modeling and Optimization Project
Reproducing results of [Optimization of area under a delay constraint in digital filter synthesis using SAT-based integer linear programming](https://ieeexplore.ieee.org/document/1688880)

## Setup
package management for this project is done via
[uv](https://docs.astral.sh/uv/).

To instantiate the virtual environment run:
```
uv sync
```
